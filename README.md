# Codecrafters Git Project

This is a complete solution for the Codecrafters build your own Git challenge.
Functions that have been implemented:
- cat-file
- checkout
- clone
- commit-tree
- hash-object
- ls-remote
- ls-tree
- tag
- unpack-objects
- write-tree

All tests are currently passing.

I may or may not continue implementing additional functions of Git, but I have no desire to do so at this moment.
